  - 特殊字符比例检测
  - 重复字符攻击防护

#### 2. ✨ 加载动画优化
- **来源**: chat-gpt-tarot-divination
- **文件**: `frontend/css/components.css`
- **功能**:
  - 弹性 cubic-bezier 动画
  - 紫金渐变边框
  - 发光阴影效果
  - Sparkle（✨）脉冲动画

#### 3. 📤 分享卡片功能
- **来源**: usongon/ai-tarot-reader
- **文件**: `frontend/js/utils/share.js`
- **功能**:
  - html2canvas 生成精美卡片
  - 下载 PNG 图片
  - 复制到剪贴板
  - 社交分享对话框

### Phase 3 - 高级功能（4个）

#### 4. 🌟 占星学元数据
- **来源**: tarot-api 生态
- **文件**: `backend/data/tarot_deck.py`
- **功能**:
  - 大阿尔卡纳：行星、星座对应
  - 小阿尔卡纳：元素、季节、方位
  - 数字牌：数秘学含义
  - 完整的占星学信息体系

**数据示例**:
```python
# 大阿尔卡纳
"愚者": {"planet": "天王星", "element": "风"}
"魔术师": {"planet": "水星", "element": "风"}

# 花色属性
"wands": {
  "element": "火", "season": "夏季",
  "direction": "南方", "time": "正午"
}

# 数字含义
"ace": {"number": 1, "meaning": "开始、潜力、种子"}
```

#### 5. 🎨 主题切换
- **来源**: chat-gpt-tarot-divination
- **文件**: `frontend/css/themes.css`, `frontend/js/utils/theme.js`
- **功能**:
  - 亮色/暗色主题无缝切换
  - 右上角 ☀️/🌙 浮动按钮
  - localStorage 持久化偏好
  - 平滑过渡动画

**主题对比**:
| 元素 | 暗色主题 | 亮色主题 |
|------|----------|----------|
| 背景 | #0a0a0f | #f5f5f7 |
| 卡片 | #1a1a2e | #ffffff |
| 主文字 | #e0e0e0 | #1f2937 |
| 强调色 | 紫金配色 | 相同 |

#### 6. 🧠 智能牌阵推荐
- **来源**: AI-Tarot-Reader
- **文件**: `backend/services/spread_recommender.py`
- **功能**:
  - AI 分析问题复杂度
  - 推荐最合适牌阵
  - 显示推荐理由
  - 置信度评分（0-100%）

**工作流程**:
```
用户输入问题
    ↓
点击「🤖 智能推荐」
    ↓
后端 Claude API 分析
    ↓
返回推荐牌阵 + 理由
    ↓
自动选择牌阵
```

**API 端点**: `POST /api/reading/recommend-spread`

#### 7. 🖼️ 塔罗牌图片
- **来源**: tarot-api 生态
- **文件**: `backend/data/card_images.py`
- **功能**:
  - 78张 Rider-Waite 塔罗牌图片
  - 图片悬停显示信息覆盖层
  - 加载失败自动降级到文字版本
  - 正逆位图片旋转

**图片源**: Sacred-Texts.com 公开塔罗牌图片

## 💡 技术亮点

### 1. 多层输入验证
```python
用户输入 → sanitize_input() → validate_question() → 业务逻辑
         清理控制字符      多层安全检测        安全的输入
```

### 2. AI 驱动的智能推荐
```python
Claude API 分析
    ↓
问题复杂度 + 时间维度 + 洞察深度
    ↓
最优牌阵 + 理由 + 置信度
```

### 3. 主题系统架构
```javascript
localStorage ↔ theme.js ↔ CSS variables
   持久化      状态管理     视觉呈现
```

### 4. 图片降级策略
```
加载图片 → 成功显示
    ↓
  失败?
    ↓
自动降级到文字版本
```
