# 🎨 布局设计说明

## 三栏布局概览

新版本采用三栏式布局，实现一屏内展示所有内容，无需滚动。

```
┌─────────────────────────────────────────────────────────────┐
│                    🌙 AI 塔罗占卜 🌙                          │
│                  探索命运的神秘指引                            │
├─────────────────┬─────────────────┬───────────────────────────┤
│                 │                 │                           │
│  🔮 你的问题     │  🃏 抽取的牌     │    ✨ 塔罗解读             │
│                 │                 │                           │
│  ┌───────────┐  │  ┌─────────┐   │   ┌─────────────────┐    │
│  │           │  │  │         │   │   │                 │    │
│  │  问题输入  │  │  │  卡牌   │   │   │   AI 流式解读    │    │
│  │  文本框   │  │  │  展示   │   │   │   (打字机效果)   │    │
│  │           │  │  │  区域   │   │   │                 │    │
│  └───────────┘  │  └─────────┘   │   │   - 整体概览     │    │
│                 │                 │   │   - 牌面解读     │    │
│  [开始占卜]     │                 │   │   - 组合分析     │    │
│                 │                 │   │   - 行动建议     │    │
│                 │                 │   └─────────────────┘    │
│  (1/3 宽)       │  (1/3 宽)       │      (2/3 宽)            │
└─────────────────┴─────────────────┴───────────────────────────┘
```

## 布局特点

### 桌面端（>1024px）- 三栏布局
- **左上（1/3）**: 问题输入区
  - 文本框占满区域
  - "开始占卜"按钮在底部
  
- **左下（1/3）**: 塔罗牌展示区
  - 卡牌垂直居中显示
  - 单牌阵：1张卡牌
  - 三牌阵：3张卡牌垂直排列
  
- **右侧（2/3）**: AI 解读区
  - 占据整个右半部分
  - 流式输出，打字机效果
  - 自动滚动到最新内容

### 平板端（768px-1024px）- 两栏布局
```
┌──────────────┬─────────────────┐
│  问题输入    │                 │
├──────────────┤   AI 解读区     │
│  卡牌展示    │  (占据右侧)     │
└──────────────┴─────────────────┘
```

### 移动端（<768px）- 单栏堆叠
```
┌─────────────────┐
│   问题输入       │
├─────────────────┤
│   卡牌展示       │
├─────────────────┤
│   AI 解读        │
└─────────────────┘
```

```css
#question-section { grid-column: 1; }

```

## 空状态设计

每个区域在未使用时显示优雅的空状态提示：

- **问题区**: 占位符文本提示
- **卡牌区**: 🌟 "开始占卜后，卡牌将在这里显示"
- **解读区**: 🔮 "AI 解读将在这里显示"

## 优势

### 1. 一屏展示
- ✅ 无需滚动即可看到所有信息
- ✅ 提升用户体验
- ✅ 更专业的布局

### 2. 视觉层次清晰
- 左侧：输入和结果
- 右侧：详细解读
- 信息流向自然

- 解读区域获得最大空间
- 卡牌展示区域刚好合适

### 4. 响应式适配
- 大屏：三栏并排
- 平板：两栏布局
- 手机：单栏堆叠

## 关键技术

### Flexbox + Grid 混合
- Grid: 主布局（三栏）
- Flexbox: 内部布局（垂直居中）

### 高度控制
```css
.app-container {
  height: 100vh;          /* 全屏高度 */
  display: flex;
  flex-direction: column;
}

main {
  flex: 1;                /* 占据剩余空间 */
  min-height: 0;          /* 防止溢出 */
}
```

### 滚动控制
- 问题区：textarea 自动调整
- 卡牌区：居中展示，不滚动
- 解读区：overflow-y: auto
