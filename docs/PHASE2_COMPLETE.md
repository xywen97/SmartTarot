### 3. 卡牌 3D 翻转动画 ⭐⭐⭐⭐

**功能描述：**
卡牌抽取时的精美 3D 翻转效果。

**动画特点：**
- 🎬 **3D 透视** - perspective: 1000px
- 🎬 **翻转效果** - rotateY(180deg)
- 🎬 **顺序播放** - 每张牌延迟 300ms
- 🎬 **平滑过渡** - 0.6s ease-in-out

**视觉效果：**
1. 初始状态：显示卡牌背面（🌙 月亮图标）
2. 翻转动画：180度旋转
3. 最终状态：显示卡牌正面内容

**实现原理：**
```css
.card-flip-container {
  perspective: 1000px;
}

.card-flip-inner {
  transform-style: preserve-3d;
  transition: transform 0.6s;
}

@keyframes flip3d {
  0% { transform: rotateY(0deg); }
  50% { transform: rotateY(90deg); }
  100% { transform: rotateY(180deg); }
}
```

**代码位置：**
- `frontend/css/components.css` - 3D动画样式
- `frontend/js/ui/card.js` - 卡牌渲染逻辑

**时序控制：**
- 第1张牌：0ms 后翻转
- 第2张牌：300ms 后翻转
- 第3张牌：600ms 后翻转
- 以此类推...
