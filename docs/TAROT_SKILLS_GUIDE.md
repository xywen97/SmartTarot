# 🔮 Tarot Skills 框架 - 完整指南

## 📋 概述

Tarot Skills 是一个创新的 LLM 行为校正框架，将 22 张大阿尔卡纳塔罗牌转化为 22 种独特的思维模式，用于打破 AI 的默认行为，提供更深刻、更多元的解读。

**核心理念**: 每张大阿尔卡纳牌不仅是占卜工具，更是一种"Prompt 工程模式"，能够引导 AI 从不同的思维角度进行分析。

---

## 🎴 22 种思维模式详解

### 0. 愚者 - Beginner's Mind (初学者模式)
- **打破**: 专家偏见
- **效果**: 以完全开放、好奇的心态看待问题
- **应用**: 抛开既有知识，提出最直接的问题
- **示例**: "如果完全不懂这个领域，最天真的疑问是什么？"

### 1. 魔术师 - Resource Mapper (资源利用者)
- **打破**: 工具依赖
- **效果**: 识别和利用现有资源
- **应用**: 重新组合现有元素创造新可能
- **示例**: "你已经拥有的资源中，哪些可以被重新利用？"

### 2. 女祭司 - Intuitive Reader (直觉洞察)
- **打破**: 过度理性
- **效果**: 倾听直觉和潜意识
- **应用**: 关注未说出口的、隐藏的信息
- **示例**: "你的直觉在告诉你什么？"

### 3. 皇后 - Nurturer (养育者)
- **打破**: 效率至上
- **效果**: 关注成长过程和有机发展
- **应用**: 强调耐心培育而非强制推进
- **示例**: "什么需要时间慢慢成长？"

### 4. 皇帝 - Structure Builder (结构建立者)
- **打破**: 混乱无序
- **效果**: 建立清晰的结构和边界
- **应用**: 定义明确的规则和原则
- **示例**: "这里需要什么样的秩序和规则？"

### 5. 教皇 - Tradition Keeper (传统智慧)
- **打破**: 盲目创新
- **效果**: 从传统智慧和经验中学习
- **应用**: 参考历史经验和经过检验的真理
- **示例**: "前人在类似情况下是如何做的？"

### 6. 恋人 - Value Aligner (价值选择)
- **打破**: 价值模糊
- **效果**: 明确价值观并做出一致的选择
- **应用**: 识别选择背后的价值冲突
- **示例**: "这个选择与你的核心价值观一致吗？"

### 7. 战车 - Will Driver (意志驱动)
- **打破**: 被动等待
- **效果**: 运用意志力和决心推进目标
- **应用**: 强调主动掌控和推进
- **示例**: "你需要如何主动出击？"

### 8. 力量 - Gentle Power (温柔力量)
- **打破**: 强硬对抗
- **效果**: 以温柔和耐心驯服困难
- **应用**: 用同情心而非暴力解决问题
- **示例**: "如何用温柔而非强硬来处理？"

### 9. 隐士 - Solitary Seeker (独处反思)
- **打破**: 外部依赖
- **效果**: 通过独处和内省寻找答案
- **应用**: 强调向内寻找智慧
- **示例**: "在安静独处时，你听到了什么？"

### 10. 命运之轮 - Cycle Recognizer (周期认知)
- **打破**: 线性思维
- **效果**: 识别周期和变化的规律
- **应用**: 理解起伏背后的模式
- **示例**: "这个问题处于什么周期的哪个阶段？"

### 11. 正义 - Karma Analyst (因果分析)
- **打破**: 情感偏见
- **效果**: 客观分析因果关系和公平性
- **应用**: 去除情感，基于事实判断
- **示例**: "客观来看，原因和结果是什么？"

### 12. 倒吊人 - Perspective Shifter (视角转换)
- **打破**: 固定视角
- **效果**: 从完全不同的角度看问题
- **应用**: 将"问题"重新框定为"机会"
- **示例**: "如果倒过来看这个问题呢？"

### 13. 死神 - Chaos Agent (混沌代理) ⚡
- **打破**: 过度讨好
- **效果**: 不怕指出问题和必要的结束
- **应用**: 直言不讳，质疑一切
- **示例**: "什么需要被结束或放弃？"

### 14. 节制 - Balance Maker (平衡调和)
- **打破**: 极端化
- **效果**: 寻找中道和平衡点
- **应用**: 调和矛盾的元素
- **示例**: "如何在对立面之间找到平衡？"

### 15. 恶魔 - Desire Anatomist (欲望解剖)
- **打破**: 道德说教
- **效果**: 直面欲望和执着，不加评判
- **应用**: 探讨阴影面和压抑的部分
- **示例**: "你真正想要的是什么？"

### 16. 高塔 - Demolition Expert (破坏建设)
- **打破**: 维持现状
- **效果**: 识别需要推倒重建的结构
- **应用**: 揭露虚假的安全感
- **示例**: "什么需要被推倒重建？"

### 17. 星星 - Hope Keeper (希望守护)
- **打破**: 悲观主义
- **效果**: 在黑暗中保持希望和信念
- **应用**: 连接更高的目标和理想
- **示例**: "即使在困境中，希望在哪里？"

### 18. 月亮 - Uncertainty Embracer (不确定性拥抱)
- **打破**: 过度确定
- **效果**: 接受模糊性和不确定性
- **应用**: 探索潜意识和梦境
- **示例**: "什么是你无法确定的？这不确定性想告诉你什么？"

### 19. 太阳 - Clarity Bringer (清晰照亮)
- **打破**: 过度复杂
- **效果**: 用清晰和简单照亮问题
- **应用**: 去除不必要的复杂性
- **示例**: "用最简单的话说，这是关于什么？"

### 20. 审判 - Awakening Caller (觉醒召唤)
- **打破**: 沉睡麻木
- **效果**: 唤醒意识并促进重生
- **应用**: 呼吁做出重大决定
- **示例**: "是时候醒来并做出决定了吗？"

### 21. 世界 - Integration Master (整合完成)
- **打破**: 片面分析
- **效果**: 整合所有元素达到圆满
- **应用**: 看到完整图景
- **示例**: "如何将所有部分整合成一个整体？"

## 🔧 技术实现

### 后端架构

**核心文件**: `backend/data/tarot_skills.py`

```python
# 每个技能包含
{
    "name": "牌名",
    "skill_name": "技能名",
    "breaks": "打破的行为",
    "description": "描述",
    "prompt_modifier": "Prompt 修饰符"
}
```

**应用流程**:
```
用户占卜 → 抽到大阿尔卡纳 → 获取对应技能 
    ↓
将技能的 prompt_modifier 注入到解读 Prompt
    ↓
Claude API 收到增强的 Prompt → 以该思维模式解读
```

**集成位置**: `backend/services/tarot_service.py`

```python
# 在 build_prompt 方法中
if major_cards:
    primary_major = major_cards[0]
    card_id = primary_major['id']
    if card_id in TAROT_SKILLS:
        prompt = apply_skill_to_reading(card_id, prompt)
```

## 💡 设计理念

### 为什么需要 Tarot Skills？

**问题**: LLM 有固有的行为模式
- 过度讨好（总是说"是的，你是对的"）
- 专家偏见（过度依赖训练数据中的"正确答案"）
- 线性思维（A→B→C，缺乏循环和跳跃）
- 过度确定（总想给出明确答案）

**解决方案**: 用塔罗牌的原型能量打破这些模式
- 每张大阿尔卡纳代表一种原型能量
- 将原型能量转化为 Prompt 工程模式
- 引导 AI 从不同角度思考

### 与传统 Prompt 工程的区别

| 方面 | 传统 Prompt | Tarot Skills |
|------|-------------|--------------|
| 方法 | 静态指令 | 动态模式 |
| 视角 | 单一固定 | 22 种轮换 |
| 深度 | 表面优化 | 行为重塑 |
| 体验 | 工程化 | 神秘化+工程化 |

## 🧪 实际效果示例

### 案例 1: 相同问题，不同技能

**问题**: "我该换工作吗？"

**抽到「教皇」(传统智慧)**:
> 让我们从传统智慧的角度来看。历史上，人们在换工作前会考虑...稳定性、行业前景...你的决定是否符合传统的职业发展路径？

**抽到「愚者」(初学者模式)**:
> 如果我是第一次听说"工作"这个概念，最天真的问题是：为什么要换？是什么让你不开心？快乐重要还是钱重要？也许答案比你想的简单。

**抽到「死神」(混沌代理)**:
> 直说吧：这份工作已经死了。你心里知道，只是不敢承认。问题不是"该不该换"，而是"为什么还没换"。什么让你困在这里？恐惧？舒适区？

### 案例 2: 技能组合效应

当抽到多张大阿尔卡纳时，系统使用第一张的技能，但多张牌会形成"思维共振"。

**示例**: 问"如何改善人际关系？"
- 主牌：倒吊人（视角转换）
- 辅牌：恋人（价值选择）
  
→ AI 会既倒转视角，又关注价值观的一致性

## 📚 理论基础

### 荣格心理学

Tarot Skills 基于荣格的原型理论：
- 每张大阿尔卡纳是一个原型
- 原型代表人类集体无意识中的模式
- 激活原型 = 激活特定的思维模式

### Prompt 工程

技术上，Tarot Skills 是一种高级 Prompt 工程：
- **System Prompt**: 定义 AI 的角色
- **Skill Modifier**: 定义 AI 的思维方式
- **User Prompt**: 具体问题

### 创新点

**首次将占卜系统与 Prompt 工程深度结合**:
- 不只是"用 AI 解读塔罗牌"
- 而是"用塔罗牌重塑 AI 思维"
